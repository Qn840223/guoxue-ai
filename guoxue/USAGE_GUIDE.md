# 使用指南

## 📚 功能使用说明

### 1. 古文翻译

**使用场景**：
- 阅读古籍时遇到难懂的句子
- 学习文言文课文
- 研究历史文献

**使用步骤**：
1. 进入"古文翻译"页面
2. 在输入框中输入文言文
3. 点击"开始翻译"按钮
4. 查看翻译结果

**示例**：
```
输入：学而时习之，不亦说乎？
输出：学习知识并且按时温习，不也是很愉快的事吗？
```

**提示**：
- 可以输入整段古文
- 支持添加注释功能
- 翻译会保留原文的文学美感

---

### 2. 诗词创作

**使用场景**：
- 学习诗词创作
- 获取创作灵感
- 体验古典文学

**使用步骤**：
1. 进入"诗词创作"页面
2. 输入创作主题（如：春天、思乡）
3. 选择诗词格式（五言/七言绝句、律诗）
4. 可选：设置情感基调
5. 点击"开始创作"
6. 查看生成的诗词

**支持格式**：
- 五言绝句（4句，每句5字）
- 七言绝句（4句，每句7字）
- 五言律诗（8句，每句5字）
- 七言律诗（8句，每句7字）

**示例**：
```
主题：春天
格式：七言绝句

输出：
《春天》
春风拂柳绿如烟，
细雨润花红满园。
燕子归来寻旧巢，
莺歌燕舞乐无边。
```

---

### 3. 对联生成

**使用场景**：
- 春节贴对联
- 学习对联创作
- 文化活动

**使用步骤**：
1. 进入"对联生成"页面
2. 输入上联
3. 点击"生成下联"
4. 查看生成的下联和横批

**对联规则**：
- 字数相同
- 词性相对
- 平仄协调
- 意境相合

**示例**：
```
上联：春回大地千山秀
下联：日照神州万木荣
横批：春满人间
```

**提示**：
- 可以点击示例快速填充
- 上联字数不宜过长
- 注意平仄搭配

---

### 4. 典故查询

**使用场景**：
- 查找成语典故
- 了解历史人物
- 学习历史事件

**使用步骤**：
1. 进入"典故查询"页面
2. 输入关键词
3. 选择查询类型（典故/人物/事件）
4. 点击"开始查询"
5. 查看详细信息

**查询类型**：
- **典故**：成语、俗语的来源和含义
- **历史人物**：生平、成就、名言
- **历史事件**：背景、过程、影响

**示例**：
```
关键词：卧薪尝胆
类型：典故

输出：
出处：《史记·越王勾践世家》
故事：春秋时期，越王勾践被吴王夫差打败后...
含义：形容人刻苦自励，发奋图强。
```

### 5. 智能断句

**使用场景**：
- 整理古籍文献
- 辅助古文阅读
- 学术研究

**使用步骤**：
1. 进入"智能断句"页面
2. 输入无标点的古文
3. 点击"开始断句"
4. 查看添加标点后的文本

**示例**：
```
输入：子曰学而时习之不亦说乎有朋自远方来不亦乐乎
输出：子曰：学而时习之，不亦说乎？有朋自远方来，不亦乐乎？
```

**提示**：
- 支持长文本断句
- 自动识别句子结构
- 正确使用标点符号

---

### 6. 智能问答

**使用场景**：
- 学习国学知识
- 解答疑问
- 深入探讨

**使用步骤**：
1. 进入"智能问答"页面
2. 在输入框中输入问题
3. 点击"发送"或按回车
4. 查看AI回复
5. 可以继续追问

**问题示例**：
- "《论语》中有哪些关于学习的名言？"
- "请解释一下'仁义礼智信'的含义"
- "唐诗和宋词有什么区别？"
- "孔子的教育思想是什么？"

**提示**：
- 支持多轮对话
- 可以点击建议问题快速提问
- 回答专业且易懂

## 💡 使用技巧

### 提高翻译质量
1. 输入完整的句子或段落
2. 保持原文的标点符号
3. 如需注释，勾选"添加注释"选项

### 创作更好的诗词
1. 主题要具体明确
2. 选择合适的格式
3. 可以多次生成，选择最佳结果

### 获得准确的查询结果
1. 使用准确的关键词
2. 选择正确的查询类型
3. 可以尝试不同的关键词

### 智能问答技巧
1. 问题要清晰具体
2. 可以分步骤提问
3. 利用上下文继续深入

## 🔧 常见问题

### Q: 翻译结果不准确怎么办？
A: 
1. 检查输入的古文是否完整
2. 尝试分段翻译
3. 使用更强大的模型（如GPT-4）

### Q: 诗词不符合格律？
A: 
1. 多次生成，选择最佳结果
2. 主题要明确具体
3. 可以手动调整

### Q: 查询不到典故？
A: 
1. 检查关键词是否正确
2. 尝试不同的查询类型
3. 使用更通用的关键词

### Q: 服务响应慢？
A: 
1. 检查网络连接
2. 确认API密钥有效
3. 考虑使用国内AI服务

### Q: 如何保存结果？
A: 
1. 复制文本内容
2. 截图保存
3. 未来版本将支持历史记录
