# 🚀 快速部署指南

## 最简单的方法：Vercel 一键部署

### 步骤1：准备GitHub账号

如果没有GitHub账号：
1. 访问：https://github.com
2. 点击 "Sign up"
3. 填写邮箱、密码
4. 验证邮箱

### 步骤2：创建GitHub仓库

1. 登录GitHub
2. 点击右上角 "+" → "New repository"
3. 填写信息：
   - Repository name: `guoxue-ai`
   - Description: `国学AI助手`
   - Public（公开）
   - 勾选 "Add a README file"
4. 点击 "Create repository"

### 步骤3：上传文件

1. 在仓库页面，点击 "Add file" → "Upload files"
2. 拖拽以下文件到页面：
   - `guoxue-web.html`
   - `index.html`
   - `vercel.json`
3. 点击 "Commit changes"

### 步骤4：部署到Vercel

1. 访问：https://vercel.com
2. 点击 "Sign Up"
3. 选择 "Continue with GitHub"
4. 授权Vercel访问GitHub
5. 点击 "Import Project"
6. 选择 `guoxue-ai` 仓库
7. 点击 "Deploy"
8. 等待1-2分钟

### 步骤5：获取网址

部署完成后，你会看到：
```
🎉 Congratulations!
Your project is live at:
https://guoxue-ai.vercel.app
```

**复制这个网址，分享给所有人！**

## 💡 自定义域名（可选）

如果你有域名（如 `guoxue.com`）：

1. 在Vercel项目中，点击 "Settings"
2. 点击 "Domains"
3. 输入你的域名
4. 按照提示配置DNS
5. 等待生效
