## 🚀 如何使用

### 最快方式（3步）

1. **获取API密钥**
   - 访问 DeepSeek 或 OpenAI
   - 注册并创建API密钥

2. **配置环境**
   ```bash
   cd backend
   copy .env.example .env
   # 编辑 .env 填入API密钥
   ```

3. **启动应用**
   ```bash
   双击 start.bat
   ```

### 访问应用
- 前端：http://localhost:3000
- API文档：http://localhost:8000/docs
