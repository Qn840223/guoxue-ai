# 🌐 国学AI助手 - 网页版使用指南

## 🎉 欢迎使用网页版！

这是一个**纯前端版本**，无需安装任何后端服务，直接在浏览器中使用！

---

## 🚀 两种使用方式

### 方式一：独立网页版（最简单）

**只需一个HTML文件，双击即可使用！**

1. **打开文件**
   ```
   双击 frontend/index-standalone.html
   ```

2. **配置API密钥**
   - 在页面顶部的"API配置"区域
   - 输入你的DeepSeek或OpenAI API密钥
   - 选择对应的API地址

3. **开始使用**
   - 选择功能标签（古文翻译、诗词创作等）
   - 输入内容
   - 点击按钮即可

**就这么简单！** ✨

---

### 方式二：完整前端应用

**需要Node.js，但功能更完整，界面更美观**

1. **安装依赖**
   ```bash
   cd frontend
   npm install
   ```

2. **配置API密钥**
   
   编辑 `frontend/.env` 文件：
   ```env
   VITE_API_KEY=your_deepseek_api_key_here
   VITE_API_BASE_URL=https://api.deepseek.com/v1
   VITE_MODEL_NAME=deepseek-chat
   ```

3. **启动应用**
   ```bash
   npm run dev
   ```

4. **访问应用**
   
   打开浏览器访问：http://localhost:3000

## 🔑 获取API密钥

### DeepSeek（推荐国内用户）

1. 访问：https://platform.deepseek.com
2. 注册账号
3. 进入"API Keys"页面
4. 点击"创建新密钥"
5. 复制密钥（格式：`sk-xxxxxxxx`）
6. 充值10-20元即可使用很久

**优点：**
- ✅ 国内访问快
- ✅ 价格便宜
- ✅ 性能优秀

### OpenAI（国际用户）

1. 访问：https://platform.openai.com/api-keys
2. 注册账号
3. 创建API密钥
4. 复制密钥

**优点：**
- ✅ 性能强大
- ✅ 功能丰富

## ❓ 常见问题

### Q: 独立网页版和完整版有什么区别？

A: 
- **独立网页版**：单个HTML文件，双击即用，适合快速体验
- **完整前端版**：需要Node.js，界面更美观，功能更完整

### Q: API密钥安全吗？

A: 
- 独立网页版：密钥只在浏览器中使用，不会上传到任何服务器
- 完整前端版：密钥存储在本地环境变量中
- 建议：不要在公共电脑上保存密钥

### Q: 为什么推荐DeepSeek？

A:
- 国内访问速度快
- 价格便宜（比OpenAI便宜很多）
- 性能优秀
- 支持中文更好

### Q: 可以离线使用吗？

A: 
- 不可以，需要联网调用AI API
- 如果需要完全离线，请使用完整版（后端+前端）配合本地模型
